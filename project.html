project Details

Project on “A DIGITAL SECURITY SYSTEM FOR DOOR LOCK SYSTEM USING RFID TECHNOLOGY ”The main objective of this project is to design and implement a digital security system which can deploy in secured zone where only authentic person can be entered. We implemented a security door locking system RFID, Radio Frequency Identification is an inexpensive technology, can be implemented for several applications such as security, asset tracking, people tracking, inventory detection, access control applications using passivee of RFID which can activate, authenticate, and validate the user and unlock the door in real time for secure access

Advantages :

  • security is high

  • Easy to access

  • Reconfigurable and reusable

  • portability
